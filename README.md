Experiência Prática I — Fase 1

Este projeto foi desenvolvido como parte da disciplina de Experiência Prática I, com foco na aplicação dos fundamentos de HTML5, incluindo semântica, formulários e multimídia.

Objetivo da Fase 1:
Construir a base estrutural da plataforma utilizando HTML5 semântico, com páginas organizadas, formulários interativos e recursos multimídia.

Páginas Desenvolvidas:

- `index.html`— Página inicial com informações sobre a organização e seus pilares de atuação (educação, alimentação e saúde).
- `projects.html` — Página de projetos sociais, com filtros por área (educação, alimentação, saúde) e cards interativos.
- `cadastro.html` — Página de cadastro com formulário completo para novos voluntários.
- `contact.html` — Página de contato com formulário para envio de dúvidas e sugestões.
- `about.html` — Página adicional com informações complementares sobre a ONG.

Recursos Implementados:

- Estrutura semântica com uso de `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- Hierarquia de títulos aplicada corretamente (`<h1>` a `<h4>`)
- Imagens incluídas nas páginas via link externo e pasta `/imagens`
- Formulário completo em `cadastro.html` com campos:
- Nome Completo, E-mail, CPF, Telefone, Data de Nascimento, Endereço, CEP, Cidade e Estado
- Validação nativa com atributos HTML5 (`required`, `type`, `pattern`)
- Agrupamento lógico com `<fieldset>` e `<legend>`
- Máscaras de input aplicadas via JavaScript
- Formulário adicional em `contact.html` com campos para nome, e-mail e mensagem
- Filtros interativos em `projects.html` para exibir projetos por categoria
- Modal de detalhes para cada projeto (Educar, Alimentar, Cuidar)

Tecnologias Utilizadas:

- HTML5
- CSS3 (com arquivos separados: `variables.css`, `layout.css`, `components.css`, `responsive.css`, `style.css`)
- JavaScript (`script.js`, `templates.js`, `spa.js`, `formValidation.js`)

Checklist de Entrega — Fase 1:

- [✔] Estrutura de pastas organizada (HTML, CSS, JS, imagens)
- [✔] Páginas obrigatórias implementadas: `index.html`, `projects.html`, `cadastro.html`
- [✔] Estrutura semântica e hierarquia de títulos aplicadas
- [✔] Imagens otimizadas e organizadas
- [✔] Formulário completo com validação e máscaras
- [✔] Repositório público no GitHub com todos os arquivos
